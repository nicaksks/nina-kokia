Comandos personalizados

Os comandos só pode ser criados por moderadores.

COMANDOS
Para criar um comando você vai usar: !command add [nome] [descrição]
Para editar um comando você vai usar: !command edit [nome] [descrição]
Para deletar um comando você vai usar: !command delete [nome]

Use !comandos para ver a lista dos comandos.

VARIÁVEL DISPONÍVEL
Kokia possui algumas variáveis para serem colocada nos comandos.

(_count_) - é a variável de contagem. Sempre que um comando que possui essa variável for executado, será adicionado +1 no contador.

!command add teste - Nica deu sorte no RNG (_count_) veze(s).
!teste - Nica deu sorte no RNG 1 veze(s).
